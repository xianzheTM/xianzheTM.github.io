String

String内部是一个private final byte[] value,故它是不可变的。以前是char数组，现在改了。

1.字面量是在方法区的字符串常量池

2.new出来的在堆中

3.字面量拼接的结果是在常量池

4.拼接时若涉及到变量，那就是在堆

5.intern()方法是“池化”的意思，将某字符串“放到”常量池。

具体来说：堆中有“abcd”字符串对象，调用这个方法后会在常量池中找有没有这个字面量，若有，则返回该字面量引用，若没有，则复制一份放到常量池。在jdk1.7之后，不再这样做。因为从堆中复制对象到方法区并不快速，而且常量池放太多会引发溢出错误。1.7后是这样处理的：若是没有则将字符串对象的引用放到常量池。另外，1.7时字符串常量池放到了堆中。后来1.8又放回去了，只不过这是方法区不再是方法区，而是元空间。

基本数据类型，包装类–>String：用String的valueOf(xxx)方法

String–>基本数据类型，包装类:用包装类的静态方法：parseXxx(str)

String–>char数组：String的toCharArray()

char数组–>String：用String的构造器

String–>byte数组：用String的getBytes(),可以使用默认的字符集，也可以指定字符集

byte数组–>String：用String的构造器

其他相关类

StringBuffer效率低，线程安全，可变的。

StringBuilder效率高，线程不安全，可变的。

二者都是继承于AbstractStringBuilder，这个抽象类用byte[] value存储字符串，故是可变的。

二者内部数组默认长度是16，new StringBuilder()长度为16，new StringBuilder("abc")，长度为19.但是用length方法结果是0和3，即统计字符数。

若容量不足，就会扩容。默认会扩大为原长度*2+2，然后将原数组拷贝进去。

故建议在用构造器时指定容量，避免扩容。

日期时间API

JDK8之前

System.currentTimeMillis():1970-1-1到现在的毫秒数。

Date类：

java.util.Date类：

Date date = new Date();
System.out.println(date.toString());//Wed May 27 20:36:03 CST 2020
System.out.println(date.getTime());//1590583085383,1970-1-1到现在的毫秒数
Date date1 = new Date(15908530064L);//long型的毫秒数，date的构造器大部分已过时，除了这个和空参构造器
System.out.println(date1);//Sat Jul 04 11:02:10 CST 1970

java.sql.Data类：对应于数据库中的日期类型变量,而且它是util.data的子类。

java.sql.Date date2 = new java.sql.Date(12345665412L);
System.out.println(date2);//1970-05-24

以上两种互相转换：多态和强转（new了一个子类，利用多态给父类，再强转为子类），如果要util.date直接转为sql.date，即一个真正的父类（不是子类多态后的父类）转为子类，那么让util.date的getTime方法作为sql.date的构造器参数即可。

java.text.SimpleDateFormat类：对Date类的格式化和解析

两个操作：

格式化：日期–>字符串

解析：字符串–>日期

Date date = new Date();
System.out.println(date.toString());

SimpleDateFormat sdf = new SimpleDateFormat();
String format = sdf.format(date);
System.out.println(format);//2020/5/28 上午11:18

try {
        Date date1 = sdf.parse("2020/5/28 上午11:18");
        System.out.println(date1);//Thu May 28 11:18:00 CST 2020
    } catch (ParseException e) {
        e.printStackTrace();
    }

还可以在构造器中加入参数，指定格式。

DateFormat dateFormat = new SimpleDateFormat("dd-MM-yy:HH:mm:ss");
Date date = new Date();
String dateStr = dateFormat.format(date);
System.out.println(dateStr);//28-05-20:11:37:17
Date date1 = dateFormat.parse("27-06-20:11:00:00");
System.out.println(date1);//Sat Jun 27 11:00:00 CST 2020

将指定格式日期转为sql.Date：

SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-hh");
Date date = dateFormat.parse("2020-10-20");
java.sql.Date date1 = new java.sql.Date(date.getTime());
System.out.println(date1);
