Comparable

String,包装类实现了包装类接口，实现了compareTo方法，给出了比较对象的方式。默认从小打到排。

重写compareTo()的规则：如果调用者this大于形参，返回正数，否则负数。
普通对象排序只要继承接口，实现方法即可。

Comparator

若没有实现上面的接口，又不好修改代码，或者要临时换个排序方式，就用Comparator。

String[] arr = new String[]{"aa", "cc", "mm", "hh", "ll"};
Arrays.sort(arr, new Comparator<String>() {
    @Override
    public int compare(String o1, String o2) {
        if (o1 != null && o2 != null) {
            return -o1.compareTo(o2);
        }
        return 0;
    }
});

枚举类

public enum Season {
    SPRING("春天", "春暖花开"),//开头就要定义好枚举类的对象，用逗号分隔
    SUMMER("夏天", "夏日炎炎");


    private final String seasonName;
    private final String seasonDesc;

    Season(String seasonName, String seasonDesc) {
        this.seasonName = seasonName;
        this.seasonDesc = seasonDesc;
    }

    public String getSeasonName() {
        return seasonName;
    }

    public String getSeasonDesc() {
        return seasonDesc;
    }
}

枚举类的父类是java.lang.Enum，故其tostring方法已被重写。

interface show{
    void display();
}

public enum Season implements show{//枚举类实现接口
    SPRING("春天", "春暖花开"){//让每个对象有各自的表现形式，若是不想这样，就在后面实现display方法即可
        @Override
        public void display() {
            System.out.println("这是春天");
        }
    },
    SUMMER("夏天", "夏日炎炎"){
        @Override
        public void display() {
            System.out.println("这是夏天");
        }
    };


    private final String seasonName;
    private final String seasonDesc;

    Season(String seasonName, String seasonDesc) {
        this.seasonName = seasonName;
        this.seasonDesc = seasonDesc;
    }

    public String getSeasonName() {
        return seasonName;
    }

    public String getSeasonDesc() {
        return seasonDesc;
    }
}

两个方法：

Season[] values = Season.values();//返回枚举类中的对象列表
for (int i = 0; i < values.length; i++) {
    Season value = values[i];
    System.out.println(value);
}
Season spring = Season.valueOf("SPRING");//根据名字找对应的对象
System.out.println(spring);

注解

第一种：文档注解

第二种：JDK内置

@Override,（重写），@Deprecated,（已过时），@SuppressWarnings，（抑制编译器警告）

第三种：自定义注解

自定义注解自动继承java.lang.annotation.Annotation接口。

public @interface MyAnnotation {
    String value() default "hello";//内部成员通常用value,默认值用default，如果没有成员，表明是标识作用
}

第四种：元注解

对现有注解进行解释说明。

@Retention：标明注解的生命周期，具体在RetentionPolicy（枚举类）里面：SOURCE,CLASS(默认),RUNTIME(反射能够获取的)

@Target：标明注解能修饰哪些程序元素

@Document：标明Javadoc文档中显示注解

@Inherit：标明注解具有继承性

可重复注解：@Repeatable，在MyAnnotation上声明@Repeatable，成员值为MyAnnotations.class,MyAnnotation的Target和Retention必须要和MyAnnotations的相同。

类型注解：@Target注解中有个ElementType[],即使用@Target时的成员值，其中ElementType.TYPE_PARAMETER表示改注解能卸载变量的声明语句中,ElementType.TYPE_USE表示该注解能写在使用类型的任意语句中。
